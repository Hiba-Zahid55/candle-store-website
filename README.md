# Candle Store E-Commerce Website

## Overview

A full-stack MERN e-commerce website for purchasing candles online.

## Features

- User Registration & Login
- Admin Login
- Admin Dashboard
- Product Management
- Add Products
- Update Products
- Delete Products
- MongoDB Database
- REST APIs
- Cash On Delivery Orders

## Tech Stack

Frontend:
- React.js
- CSS
- JavaScript

Backend:
- Node.js
- Express.js

Database:
- MongoDB

Tools:
- Postman
